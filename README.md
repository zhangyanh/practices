#### 项目初始化
``` 项目初始化
npm install
```
#### 启动 静态资源的web服务
``` 项目初始化
npm run debug
```
#### 启动 用于调试的web服务
``` 项目初始化
node server/index
```
#### 发布前，合并压缩
``` 项目初始化
npm run prod
```
